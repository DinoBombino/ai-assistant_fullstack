# Запуск проекта

## Фронтенд

```bash
cd client
npm run dev
```

## После запуска:
```
➜  Local:   http://localhost:3000/
➜  Network: use --host to expose
➜  press h + enter to show help
```

---

## Бэкенд

```bash
cd server
npm run dev
```

## После запуска:
```
Server running on port 5000
```

---
